 How to Use

1. Clone or download this repository.  
2. Upload the website files (`index.html`, `css/`, `js/` folder) to your **Amazon S3** bucket.  
3. Enable static website hosting on your S3 bucket.  
4. Configure your **CloudFront** distribution to use the S3 website endpoint as the origin.  
5. Set proper bucket policies to allow public read access.  
6. Access your site via the CloudFront domain to enjoy fast and secure content delivery.  

---

 Pro Tips

Make sure to disable Block Public Access settings in S3 for public visibility.  
 CloudFront caches your site — changes may take some minutes to propagate globally.  
Watch the detailed walkthrough videos in the videos/ folder for step-by-step setup guidance.
